Grammar: Tools and Purpose

When we describe the purpose or function of tools in English, we use:

Structure Grammar Form Meaning / Use Example
be used for + -ing for + verb-ing General purpose A ruler is used for measuring.
be used to + verb to + base verb Specific action A ruler is used to measure.

Examples

  • A pencil is used for writing.
  • A computer is used for playing games.
  • A knife is used to cut food.
  • A key is used to open doors.
